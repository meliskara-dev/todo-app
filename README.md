# Todo App

A simple task management application built with React and Tailwind CSS.

## Features

- Add new tasks
- List all tasks
- Update existing tasks
- Delete tasks
- LocalStorage support (data persists after page refresh)

## Technologies Used

- ReactJS
- Tailwind CSS
- Vite
- LocalStorage

## Installation
```bash
npm install
npm run dev
```
